🐍 Study Python Core
A structured repository for learning and mastering Python Core fundamentals.
This project focuses on building a strong foundation through theory, examples, and practice exercises.
📌 Objectives
Understand Python syntax and core concepts
Practice problem-solving with Python
Build a solid base for advanced topics (OOP, Web, AI, etc.)
Prepare for real-world programming tasks
🧠 Topics Covered
Variables & Data Types
Operators
Conditional Statements
Loops (for, while)
Functions & Recursion
Data Structures (List, Tuple, Set, Dictionary)
File Handling
Exception Handling

Each topic includes:
Explanation

Practice exercises
Focus on understanding logic, not just syntax
